Here is a app to serve image search!

Example Usage:

  • Please attach the query string at the end of url like this:

    https://image-search-layer-masonlee.c9users.io/search/funny

  • And you can add a 'offset' id to browse the later results:

    https://image-search-layer-masonlee.c9users.io/search/funny?offset=20

  • This is a link to the recent search:

    https://image-search-layer-masonlee.c9users.io/latest
